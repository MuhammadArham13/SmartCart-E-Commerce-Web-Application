# 🛒 SmartCart – E-Commerce Web Application

SmartCart is a full-stack e-commerce web application built using Flask and MySQL. It simulates a real-world online shopping platform with product browsing and cart functionality.

## 🚀 Features
- Product listing
- Shopping cart system
- Backend with Flask
- Database integration using MySQL
- Responsive frontend using Bootstrap

## 🛠️ Tech Stack
- Python
- Flask
- MySQL
- HTML, CSS, Bootstrap
- JavaScript

## 📂 Project Structure
- `app.py` – Main backend application
- `templates/` – HTML files
- `static/` – CSS, JS, assets
- `database/` – SQL files
